32-Bit CPU Core

  • Tools Used: SystemVerilog, Synopsys VCS, UVM with SVA, Synopsys Design Compiler, and Synopsys PrimeTime
  • Github URL: Project Link

Designed and implemented a 32-bit pipelined RISC-V CPU core in SystemVerilog, integrating hazard detection and forwarding logic to handle data and control dependencies across the pipeline stages.

Developed a structured verification environment using UVM principles, SystemVerilog Assertions (SVA), and coverage metrics; simulated the design with industry-standard tools to validate instruction execution, timing behavior, and corner cases.

Prototyped the CPU on FPGA hardware using Vivado, leveraging Integrated Logic Analyzer (ILA) for real-time signal tracing and cycle-accurate debug, ensuring correct pipeline operation and reliable performance under various workloads.
